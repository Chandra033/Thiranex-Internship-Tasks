# Task 1 - Data Cleaning & Visualization

## Objective

The objective of this project is to clean, process, and visualize data using Python. The project focuses on handling missing values, removing duplicates, identifying outliers, and generating meaningful visualizations to extract insights from the dataset.

## Dataset

Titanic Dataset

## Tasks Performed

* Loaded the dataset using Pandas.
* Identified and handled missing values.
* Removed duplicate records.
* Performed data preprocessing and cleaning.
* Detected outliers using visualization techniques.
* Conducted exploratory data analysis.
* Built an interactive dashboard using Streamlit.
* Generated visual reports and insights.

## Visualizations

* Survival Distribution
* Gender Distribution
* Passenger Class Analysis
* Age Distribution
* Fare Distribution
* Outlier Detection using Box Plots

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Streamlit

## Key Findings

* Passenger survival was influenced by gender and passenger class.
* Female passengers had a higher survival rate than male passengers.
* First-class passengers had a higher chance of survival.
* The dataset contained missing values that were successfully handled during preprocessing.
* Visualizations helped identify patterns and trends in the data.
